# 📊 投资组合回测工具

**多资产组合回测 · 压力测试 · 对冲分析 · 交互式图表 · 报告导出**

---

## 简介

这是一个基于 **Streamlit** + **AKShare** 的投资组合回测 Web 应用。你可以：

- 📊 **配置多资产组合**（如"60% 沪深300 + 20% 黄金 + 20% 国债"），一键运行回测
- 📈 **查看权益曲线、回撤、年度/月度收益热力图**等交互式图表
- ⚡ **压力测试**，分析在 COVID、熊市、金融危机等极端事件下的表现
- 🛡️ **对冲分析**，对比添加对冲资产（如黄金）前后的组合表现
- 📥 **下载完整 HTML 报告**，包含所有图表和指标

## 功能特性

| 功能 | 说明 |
|------|------|
| 全球市场覆盖 | A股、港股、美股、ETF（通过 AKShare 免费数据源） |
| 交互式图表 | 基于 Plotly，支持缩放、悬停、导出 |
| 压力测试 | 预设 6 大历史事件，支持自定义时间范围 |
| 对冲分析 | 自动计算添加对冲资产后的风险收益变化 |
| 缓存机制 | 本地缓存数据，避免重复下载 |
| 报告导出 | 一键下载包含所有图表的 HTML 报告 |

## 安装

```bash
# 1. 克隆仓库
git clone https://github.com/yuhengliuleo/llm-portfolio-backtester.git
cd llm-portfolio-backtester

# 2. 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
streamlit run app.py
```

运行后浏览器会自动打开 `http://localhost:8501`。

## 使用指南

1. **输入资产配置**：在 📊 回测 标签页输入 Ticker（如 `sh000300, sh518880, sh511010`）和对应权重（如 `0.5, 0.3, 0.2`）
2. **调整设置**：前往 ⚙️ 设置 标签页配置起始日期、再平衡频率、基准指数、压力测试事件、对冲资产
3. **运行回测**：点击"🚀 运行回测"
4. **查看结果**：在多个标签页查看权益曲线、回撤、年度/月度收益、压力测试结果
5. **下载报告**：在"📥 下载报告"标签页下载完整 HTML 报告
