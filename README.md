# 📊 投资组合回测器 v5

> 本地运行的 Streamlit 应用，支持全球 160+ 资产的智能选择、多资产基准对比、对冲分析和压力测试。

## ✨ 功能特色

- 🌍 **全球资产库**：A股/港股/美股/全球指数/商品/加密货币/外汇，支持浏览和搜索
- 📊 **多资产基准**：支持多资产加权基准组合（如 60% 沪深300 + 40% 长期国债）
- ⚡ **压力测试**：2008金融危机、2020-COVID、2022熊市等 9 个预设事件
- 📈 **完整分析**：权益曲线、回撤、月度热力图、年度收益、绩效指标
- 🛡️ **对冲对比**：可添加对冲资产（如黄金），对比对冲前后效果
- 📥 **HTML报告**：一键下载完整交互式 HTML 报告
- 🔄 **自动再平衡**：支持无/每月/每季度再平衡
- 💾 **数据缓存**：自动缓存到 `data/` 目录，避免重复下载
- 🎨 **智能选资产**：浏览模式（市场→类型→标的三级）+ 搜索模式（模糊搜索）

## 🚀 快速开始

### 1. 安装依赖

```bash
cd backtest_app
pip install -r requirements.txt
```

### 2. 运行

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`

## 📡 数据源策略（瀑布式回退）

应用采用**多源瀑布式回退**策略，确保数据获取的可靠性：

| 优先级 | 数据源 | 适用范围 | 特点 |
|--------|--------|---------|------|
| 1 | 全球指数专用 | ^GSPC, ^IXIC 等 | yfinance → stooq → AKShare ETF等效 |
| 2 | 新浪财经 | A股、美股 | 国内直连，最可靠 |
| 3 | efinance（东方财富） | 全部市场 | 国内最全面 |
| 4 | NASDAQ API | 美股 | 官方免费接口 |
| 5 | AKShare | A股/港股/美股 | 东方财富后端 |
| 6 | yfinance | 全部市场 | 全球通用，国内可能受限 |
| 7 | FMP API | 全部市场 | 需 API Key |

全球指数（^GSPC, ^N225 等）使用**ETF等效法**：通过跟踪ETF（SPY, EWJ 等）获取价格数据。

## 📊 输出指标

- 累计回报 / 年化回报 / 年化波动率

- 最大回撤 / 胜率 / 盈亏比

## 🎨 图表

- **权益曲线**：组合 vs 基准（Plotly 交互式，支持缩放）
- **回撤分析**：组合与基准回撤对比
- **年度收益**：柱状图对比
- **月度热力图**：按年月显示收益分布
- **资产配置**：饼图显示权重分布
- **压力测试**：各事件下回报/回撤柱状图

## 🛠️ 使用方法

1. **配置资产**：使用搜索式下拉框选择资产，设置权重(%)
2. **调整参数**（⚙️ 设置标签页）：起始日期、再平衡频率、基准指数
3. **压力测试**：选择历史事件进行压力测试
4. **对冲分析**：可选添加对冲资产，对比对冲前后效果
5. **运行回测**：点击 🚀 运行回测
6. **导出报告**：下载 HTML 完整报告

## 🔧 环境变量（可选）
